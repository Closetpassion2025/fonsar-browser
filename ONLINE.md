Fonsar Browser does not use any first-party online services.

The upstream SmartCookieWeb services previously documented here (news feed,
Reading Mode translation and custom search suggestions) were removed from this
fork. The only network requests the browser makes are those initiated by the
user: page loads, the selected search engine / suggestions provider (Google,
DuckDuckGo, Baidu, Naver — configurable, can be disabled), ad-block filter
list downloads, and optional page translation via Google translate.goog when
the user taps Translate page in the ⋮ menu (General settings toggle).
